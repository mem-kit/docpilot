# 🚀 快速启动指南

## 立即开始使用Agent功能

### 1️⃣ 启动应用
```bash
npm start
```

### 2️⃣ 打开文档
- 从左侧文件列表选择一个Word、Excel或PPT文档
- 等待编辑器显示"✓ 已就绪"

### 3️⃣ 启用Agent
- 点击右侧聊天面板的 **"🔧 Load MCP"** 按钮
- 看到 "🤖 Agent Mode ON" 即可

### 4️⃣ 开始对话
输入以下任意指令试试：

#### Word文档
```
帮我在文档中插入一个段落
```
```
添加一些格式化文本
```
```
替换当前选中的文字
```

#### Excel表格
```
更新表格
```

#### PowerPoint
```
更新PPT幻灯片
```

## 🎯 期望效果

输入指令后，您会看到：

1. **工具执行提示**: 
   ```
   🔧 正在执行: updateParagraph
   结果: ✅ 成功
   ```

2. **友好的回复**:
   ```
   ✅ 已成功在文档中插入新段落！
   ```

3. **文档实时更新**: 内容立即出现在文档中

## 🎨 UI说明

- **绿色状态** "✓ 已就绪" = 可以开始操作
- **橙色消息** = Agent正在执行工具
- **绿色徽章** "🤖 Agent Mode ON" = Agent已启用

## 💡 小贴士

✅ **先选择文档** - 确保文档类型匹配操作  
✅ **使用清晰指令** - "插入段落" 比 "改改文档" 更好  
✅ **查看反馈** - 每次操作都有明确的成功/失败提示  
✅ **逐步尝试** - 先测试简单操作，再尝试复杂功能  

## 🔍 故障排查

**Q: 点击"Load MCP"没反应？**  
A: 检查浏览器控制台是否有错误信息

**Q: 工具执行失败？**  
A: 确保选择了正确类型的文档（Word工具需要Word文档）

**Q: Agent不调用工具？**  
A: 使用更明确的指令，如"请插入段落"或"帮我添加文本"
