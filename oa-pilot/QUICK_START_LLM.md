# 🚀 LLM 工具调用快速开始

## 立即开始测试！

您的应用已经成功集成了 LLM Function Calling 功能。现在可以通过自然语言来操作文档了！

## ⚡ 3步快速开始

### 1️⃣ 确认应用运行
访问浏览器：http://localhost:3000

### 2️⃣ 打开文档
从左侧文件列表选择任意文档（.docx, .xlsx, 或 .pptx）

### 3️⃣ 激活 AI 助手
点击右侧聊天面板的 **"🔧 Load MCP"** 按钮

看到 **"🤖 Agent Mode Active"** 就可以开始了！

## 🎯 它是如何工作的？

```
您输入: "在文档中添加一段文字：Hello"
    ↓
AI 理解: 需要调用 updateParagraph 工具
    ↓
自动执行: updateParagraph({ text: "Hello" })
    ↓
文档更新: ✅ 成功插入段落！
```

## 📺 观察执行过程

聊天窗口会显示：

1. **🔧 工具调用**
   ```
   🔧 正在执行: updateParagraph
   参数: { "text": "Hello" }
   ```

2. **✅ 执行结果**
   ```
   ✅ 执行成功
   成功插入段落: Hello
   ```

3. **🤖 AI 总结**
   ```
   已成功在文档中添加段落："Hello"
   ```

## 🔍 调试提示

### 如果没反应？
1. ✅ 确认点击了 "Load MCP" 按钮
2. ✅ 检查是否有 "🤖 Agent Mode Active" 标识
3. ✅ 等待文档完全加载（编辑器显示内容）

### 查看详细日志
按 **F12** 打开浏览器开发者工具，在 Console 标签查看执行详情

配置文件位于 `src/config.js`:

```javascript
{
  llmURL: "https://api.deepseek.com/v1",  // LLM API 地址
  llmAPIKey: "sk-...",                     // API Key
  baseURL: "http://...",                   // OnlyOffice Server
}
```
